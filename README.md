# 📊 Customer Retention & Churn Analysis

## 🎯 Objective

Analyze customer behavior to identify churn patterns, retention drivers, and lifetime trends.

---

## 📁 Dataset

This project uses multiple datasets:

* Accounts
* Subscriptions
* Churn events
* Feature usage
* Support tickets

---

## 🧠 Methodology

* Data merging from multiple sources
* Feature engineering (churn, lifetime)
* Cohort analysis
* Behavioral segmentation

---

## 📈 Key Metrics

* Retention Rate
* Churn Rate
* Customer Lifetime

---

## 🔍 Key Insights

* High churn occurs within first 30 days
* Low feature usage strongly correlates with churn
* Premium plans retain customers longer
* High support tickets indicate churn risk

---

## 💡 Recommendations

* Improve onboarding experience
* Increase feature engagement
* Enhance support quality
* Encourage long-term subscriptions
